Break Reminder

A lightweight Windows system-tray app for custom break reminders

  • Multiple concurrent reminders with custom intervals and messages
  • Native Windows toast notifications
  • Snooze functionality (5 minutes per reminder)
  • Edit reminders while running
  • System tray integration with quick actions
  • Mute (DND) toggle and Windows Focus Assist support
